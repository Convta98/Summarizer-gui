Summarizer GUI - Offline Summarization Tool

This is a lightweight, offline desktop tool for summarizing documents and websites.

Features:
- Summarize DOCX, TXT, PDF files
- Summarize from web links
- Simple graphical interface
- No internet required to run

To install:
1. Download the installer from the link on the GitHub Page or Google Drive.
2. Run the setup file and follow the on-screen instructions.
3. A desktop shortcut will be created.
